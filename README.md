# mkfog-guide

## 项目简介

mkfog-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面内容覆盖不同领域和用途，旨在提供可供参考或使用的静态页面资源。每个页面均为独立 HTML 文件，不依赖外部服务或特定域名。

## 目录说明

```
mkfog-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（如样式表、图片等）
├── index.html      # 仓库导航页（可选）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件对应一个独立页面，文件名反映内容主题。
- **assets/**：共享资源，如通用 CSS、JavaScript 或图片文件，按需引用。
- **index.html**：若存在，可作为浏览仓库页面的入口。

## 页面归档说明

- 所有页面均为纯 HTML 编写，不包含后端逻辑或动态加载。
- 页面内容以实用、教育或参考性质为主，不涉及任何商业或推广目的。
- 归档的页面可能不定期更新，部分页面可能基于常见模板或示例修改而来。
- 每个页面均可在本地直接打开，无需联网或额外配置。

## 维护与贡献

- 欢迎通过 Issues 提交页面建议或错误报告，但请注意：不接受涉及具体网站、商业推广或敏感内容的请求。
